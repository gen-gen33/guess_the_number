# 数当てゲーム

このプロジェクトはC++を使用して実装されたシンプルな数当てゲーム。
プレイヤーはランダムに生成された数値を当てるまで試行し、試行回数がスコアとして出力されます。

## **動作方法**

1. プログラムをコンパイル：
   ```bash
   g++ -o guess_game main.cpp
   ```

2. プログラムを実行：
   ```bash
   .guess_game
   ```
